## [1.0.0](https://github.com/JoanRoucoux/angular-starter-web/releases/tag/v1.0.0) (2026-07-19)

First stable release of Angular Starter Web — a ready-to-use Angular 22 starter
(standalone, zoneless, signals) for apps embedded in a host portal that owns the
global chrome.

### Highlights

* **Angular 22** — standalone components, zoneless change detection, signals throughout
* **Feature architecture** — `pages/` + `data/` + `forms/` per feature, module boundaries enforced by Sheriff, barrel-less
* **API layer** — models and Angular services generated by Orval from the OpenAPI spec, wrapped in per-feature data services owning `rxResource` state
* **Signal Forms** — typed model factory + validation schema example
* **i18n** — Transloco (en/fr) with one lazy scope per feature, preloaded global keys and translated page titles
* **Testing** — Vitest + Angular Testing Library (100% coverage), Playwright e2e with page objects
* **Tooling** — ESLint, Prettier (import & Tailwind class sorting), Husky + lint-staged + commitlint (Conventional Commits)
* **CI/CD** — GitHub Actions pipeline (format, lint, unit tests, build, e2e), actions pinned by SHA, Dependabot, automated releases via release-please
* **Theming** — shadcn/ui-style zinc tokens with `light-dark()`, theme driven by the host via `data-theme`
